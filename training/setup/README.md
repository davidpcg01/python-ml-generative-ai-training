# Setup Guide

Use this folder when you need to run the hands-on materials outside of slide review.

- `requirements.txt`: shared Python package list for local environments
- `local/production-forecasting-local-setup.md`: step-by-step local Jupyter setup
- `colab/production-forecasting-colab-guide.md`: Google Colab workflow for the production forecasting demo

If you are only reading the training content, you do not need anything in this folder.
